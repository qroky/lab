## Argument spine (shared)

- **Thesis:** agents can now *produce* almost anything; what they can't produce
  is confidence it's done right. The bottleneck moved from producing work to
  governing it.
- **Tension (exclusive knowledge):** at Finergy — a company that builds software
  for banks — almost every AI case lands on the same two bank concerns: model
  drift during execution, and controllability of result quality.
- **Insight:** treat every piece of agent work as a product with maker-checker —
  whoever specifies doesn't execute, whoever executes never accepts their own
  work, verification is blind.
- **Trade-off (open):** I did NOT build humans out of the loop; the gate stays.
  Not a framework for everyone yet — founders and regulated first.
- **Vision:** escalation rate is a metric — the system asks you less each quarter.
